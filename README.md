# Mission Kontrol 🎛️

OpenClaw Mission Control — dashboard til styring og overvågning af AI agents.

# dashboard

## Koncept

En visuel kommandocentral til OpenClaw agents med:
- **Kanban board** — INBOX → ASSIGNED → IN_PROGRESS → REVIEW → DONE
- **Real-time updates** via WebSocket
- **Agent management** — opret, overvåg og styr agents
- **Task assignment** — automatisk tildeling via tags
- **Review gates** — human approval før kritiske handlinger
